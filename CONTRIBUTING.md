# Contributing to typeAI

Welcome to **typeAI**! We are excited to have you contribute to our Intelligence Engine..

Our goal is to build a robust tool for content automation and cognitive computing, leveraging state-of-the-art deep learning frameworks and high-performance data processing libraries. This guide will help you get started quickly and propose your improvements.

## 🛠 Tech Stack

Before you begin, please ensure you are familiar with our core stack:
* **Python 3.x**: Asynchronous application core.
* **PyTorch & TensorFlow**: Hybrid neural network backend orchestration.
* **Pandas & NumPy**: Data engineering and vectorization tools.

## 📂 Project Architecture

To ensure your contributions align with our goals, please follow our structural guidelines:

* `/src/models` — Neural network definitions (PyTorch modules and TensorFlow Keras graphs).
* `/src/pipelines` — Data preparation operations handling NumPy arrays and Pandas DataFrames.
* `/config` — Framework hyperparameter mappings and hardware allocation configurations.
* `/tests` — Statistical validation frameworks and deep learning performance benchmarks.
* `/data` — Local storage for model checkpoints and pre-computed embedding metrics (do not commit to Git).
